Preguntas

  • ¿Qué es una función?

    • Es un procedimiento almacenado en un bloque de código para que se pueda reutilizar. Evita duplicar código y mejora la legibilidad del programa.

  • ¿Hasta cuantos argumentos puedes declarar en una función?

    • Teóricamente no hay límite en el número de argumentos que puedes declarar en una función, pero dependiendo del entorno de ejecución, puede haber límites prácticos. Por ejemplo, el límite podría variar si el código de Javascript se ejecuta en un navegador Chrome o uno Firefox.
